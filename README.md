# React App - Letmeask

NLW Together; Criação de App web com ReactJS para interação de pessoas em salas e acompanhamento em tempo real.

## Tecnologias

Tecnologias usadas no projeto:

- [React](https://reactjs.org)
- [Firebase](https://firebase.google.com/)
- [TypeScript](https://www.typescriptlang.org/)

## Aprendizados:
- Typescript
- Componentes
- Hooks
- Contexto de Autenticação
- Autenticação com Google
- Base de dados no Firebase

## Um passo a mais... Modificações realizadas no projeto:
- Tema Light e Dark via contexto 
- Botões de redirecionamento para salas Admin e Room
- Responsividade no visual
- Botão de Log-out na aplicação

## Rode a aplicação em sua máquina:
Clone o projeto:

```bash
$ git clone https://github.com/denistss/nlw-together.git
$ cd nlw-together
```

Siga os passos abaixo:
```bash
# Instalar as dependências
$ npm install --global yarn
# Iniciar o projeto
$ yarn start
```

A aplicação estará disponível no seu browser pela rota [http://localhost:3000](http://localhost:3000)
